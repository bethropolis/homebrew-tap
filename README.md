My personal [Homebrew](https://brew.sh) tap containing custom formulae.

## How to use

Add the tap:

```sh
brew tap bethropolis/tap
```

Install a formula:

```sh
brew install bethropolis/tap/localgo
```

## Available formulae

| Formula | Description |
| ------- | ----------- |
| `localgo` | LocalSend v2.1 protocol implementation in Go — send and receive files over LAN |
